## Configuring a Basic Spring Application
---
This exercise demonstrates how to use Spring’s Inversion of Control (IoC) container to inject dependencies between BookService and BookRepository.

 What It Does
Configures BookRepository as a dependency for BookService using setter injection.

Uses XML-based configuration (applicationContext.xml) to wire the beans.

Validates DI setup by running the LibraryManagementApplication main class.
