## Configuring a Basic Spring Application
This exercise sets up a basic Spring application for a library management system using XML-based configuration.

 What It Does
Creates a Maven-based Spring project with BookService and BookRepository beans.

Uses applicationContext.xml to configure and wire the beans.

Demonstrates how to load the Spring context and test the setup from a main class.
