# Library Management - Spring Maven Project

This project sets up the base for a **Library Management System** using **Spring Framework**. It includes essential dependencies and plugins to get started with building a scalable web application using Spring.

---

##  What This Setup Includes

1. **Maven Project**  
   A new project named `LibraryManagement` created using Maven, which simplifies dependency management and build process.

2. **Spring Dependencies**  
   - **Spring Context** – For managing beans and application context.  
   - **Spring AOP** – To handle cross-cutting concerns like logging or security.  
   - **Spring WebMVC** – To build a web-based application following MVC architecture.

3. **Maven Compiler Plugin**  
   Configured for **Java 1.8** to ensure compatibility and proper compilation.

---

##  What This Will Do

- Prepare the project structure for Spring-based development.  
- Allow creation of web controllers, services, and AOP features.  
- Provide a clean build and dependency setup for smooth development.
